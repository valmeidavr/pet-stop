# Pet Stop

Aplicação web (React + Vite) para clínicas e pet shops, emergência veterinária e babás — região demo Vale do Paraíba / Sul Fluminense.

## Desenvolvimento

```bash
npm install
npm run dev
```

Ou `npm start` (equivale a `vite`).

## Build

```bash
npm run build
npm run preview
```

## Requisitos

- Node.js 18+
